# NextJS Boilerplate

## Tech Stack

- NextJS 14
- React 18
- TypeScript
- TailwindCSS
- Prettier
- ESLint
- Lefthook

## Getting Started

```
# Install dependencies
npm install

# Initialize Git repository (Required for Lefthook)
git init

# Install git hooks
npx lefthook install

# Start development server
npm run dev
```
